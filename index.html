Why Scala?

"Why should my developers learn and use a new technology? We already have a language that works.."

We were Java engineers, but we adopted Scala to deliver applications sooner and at less cost. Scala code on averiage is about 3 times shorter than the equivalent Java one, without lossing any performance or scalability.
